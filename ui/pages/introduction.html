TiebaSpider v1.0 - bate

贴吧内容归档与管理工具

项目介绍

这是一个由语言Python开发，依赖 asyncio, httpx, PySide6 的贴吧爬虫与帖子管理工具

开发的初心是为了及时存档自己喜欢的帖子，尤其是贴吧审核越收紧的当下

核心价值：提供稳定可靠的贴吧内容本地化存档方案，避免因帖子被删除、屏蔽等导致的信息丢失。
设计理念：在保证爬取效率与准确性的同时，提供简洁直观的用户界面，降低技术使用门槛。

简要操作指南

工具分为两个核心模块：爬取与管理，分别对应内容的获取与本地帖子操作。

爬取模块

  • 在左侧导航栏选择 爬取 页面
  • 输入目标帖子的完整链接，链接可通过浏览器地址栏复制或鼠标右键菜单的复制链接功能获得
  • 根据需求勾选只看楼主选项，以过滤其他用户的回复
  • 点击 开始爬取 按钮启动抓取进程
  • 测试URL功能用于验证输入链接是否为有效的贴吧帖子地址
  • 右侧日志区域实时显示爬取状态、进度及可能的错误信息
  • 保存格式为Markdown，最终结果不包含任贴吧表情和楼中楼回复

管理模块

  • 在左侧导航栏选择 管理 页面
  • 可对已存档的帖子执行三种核心操作：增量爬取（获取新回复）、重新爬取（覆盖更新）、删除（移除存档）
  • 在帖子条目上点击右键，可调出上下文菜单，提供打开本地浏览、复制链接等操作
  • 批量模式下，可勾选多个帖子进行统一操作
  • 提供按模式、关键字搜索帖子
  • 请确保本地安装支持 Markdown 渲染的编辑器（如 VS Code、Obsidian），并将其设置为.md 格式文件的默认打开程序。
